# Расширение: Форматировать Word по шаблону

## Описание

Пользовательский блок для Puzzle RPA Studio, который автоматически приводит Word-документы к единому стилю оформления. Блок:

- очищает документ от лишних пустых абзацев, множественных пробелов и табуляций;
- применяет единый шрифт (название и размер) ко всему тексту;
- автоматически определяет заголовки по нумерации (`1.`, `1.1.`) или ключевым словам и назначает им стили «Заголовок 1» / «Заголовок 2»;
- форматирует все таблицы: видимые границы, выравнивание, заголовочная строка (жирный + заливка);
- заменяет плейсхолдеры `{{ключ}}` на значения из входного словаря.

## Требования к окружению

- Операционная система: Windows 10 / 11
- Установлен Microsoft Word 2016 или новее
- Puzzle RPA Studio 3.0.2+

## Установка расширения

1. Распакуйте архив `WordFormatterExtension.zip` в любую папку.
2. Откройте Puzzle RPA Studio → **Настройки** → **Расширения** → **Добавление**.
3. Нажмите **Выбрать** и укажите папку `WordFormatterExtension`.
4. Выберите категорию **Интеграция приложений / Word** (она уже существует).
5. Нажмите **ОК** и перезапустите Puzzle RPA Studio.

После перезапуска блок **«Форматировать Word по шаблону»** появится в тулбоксе в категории **Интеграция приложений → Word**.

## Параметры блока

| Параметр | Тип | Обязательный | По умолчанию | Описание |
|----------|-----|:------------:|:------------:|----------|
| `input_path` | string | V | — | Путь к исходному `.docx` файлу |
| `output_path` | string | X | — | Путь для сохранения результата. Если не указан - файл перезаписывается |
| `font_name` | string | X | `Calibri` | Название шрифта |
| `font_size` | int | X | `12` | Размер шрифта в пунктах (4–72) |
| `replacements` | dict | X | `{}` | Словарь замен плейсхолдеров: `{"ключ": "значение"}` |

## Пример использования в Puzzle RPA

1. Перетащите блок на рабочую область.
2. Заполните параметры:
   - `input_path`: `C:\test\input_word.docx`
   - `output_path`: `C:\test\output_word.docx` *(необязательно)*
   - `font_name`: `Times New Roman` *(необязательно)*
   - `font_size`: `12` *(необязательно)*
3. Если в документе есть плейсхолдеры `{{Дата}}`, передайте в `replacements`:
   ```json
   {"Дата": "20.05.2025", "Организация": "ООО Речное"}
   ```
4. Запустите выполнение (F5).

## Демонстрационные файлы

- `test_input_word.docx` — тестовый документ с разнородным форматированием, лишними пустыми строками, таблицей без границ и плейсхолдером `{{Дата}}`.
- `test_output_word.docx` — результат обработки блоком (эталон).

## Известные ограничения

- Работает только с форматом `.docx` (не `.doc`).
- Требуется установленный Microsoft Word (не достаточно LibreOffice).
- Автоопределение заголовков работает для нумерованных структур (`1.`, `1.1.`) и ключевых слов (Введение, Заключение и др.). Для произвольных заголовков используйте ручные стили в Word.

```
└── src/
    └── __init__.py     # вся логика форматирования
```
